		========================
		ISO8583 Messaging System
		========================

Tools needed
============
- JDK 1.6.x or above
- Eclipse (kepler in case)
- Maven   

Software Packages
=================
- utl-1.0.0.jar (https://github.com/jporras66/utilities)

How to
======
1. Import the application into Eclipse

2. Open Application

3. Select Application -> Run -> Run As -> Maven Build -> 8583core:package

4. Review : Test Result

5. Go to ./target/apidocs, open index.html (classes descriptions)
